General Administration Module

This is a simple web application that allows you to update the configuration files for components: TAALOS, ZAHEU, RMF, TABREF and CCN2 Adapter.
In this application the user chooses one of the modules, then will have displayed screens, which in addition to storing them in the database are created and configurable. Configuration files are created because launching an application involves initializing some parameters that are done before accessing the database, like connection details to the base.
